# Switch 🔄
> **Transform your unhealthy lifestyle into a Healthy Lifestyle.**

Welcome to **Switch**, a React-based application designed to help users pivot from poor habits to sustainable wellness. This project focuses on a clean, interactive UI and serves as a showcase for React.js fundamentals and modern web design.

---

## 📂 Repository Structure

This repository uses a specific branching strategy to separate the project vision from the implementation:

* **`main` (Current):** Readme file
* **[`master`](https://github.com/kashishshahr/Switch/tree/master):** Contains the **React.js Source Code**, UI components, and frontend logic.

---

## 🛠️ Tech Stack & Learning Goals

This project was built to demonstrate proficiency in:
* **React.js:** Functional components and hooks (`useState`, `useEffect`).
